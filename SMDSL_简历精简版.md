## SMDSL — 面向机器人避障的空间约束验证与自愈系统

**背景**：现有 VLA 模型直接输出机器人动作，靠近障碍物时无法解释"为什么不能走"，碰撞后不可追溯。本项目在 LLM 与底层执行器之间插入一层声明式约束中间语言（RoboIR），让大模型只做逻辑翻译、空间计算全部交给确定性数学引擎。

---

**负责工作**

- **搭建 CAD 图纸到距离场底图的完整解析管线**。支持 FloorplanQA、工业 DWG、SVG、PNG 等五种输入格式，编写了双线墙焊接、外部连通域剔除等形态学后处理模块，最终在 1981 个测试布局上零失败生成带安全膨胀代价的 A* 导航底图，单个布局平均耗时 29.9ms。

- **设计 LLM 语义编译器的护栏与容错机制**。针对大模型容易自行编造坐标的问题，在 System Prompt 中明确禁止输出数值计算，并在编译完成后增加字段校验 —— 非白名单取值直接拒绝。同时编写了三级 JSON 解析容错器和三次重试逻辑，把 API 调用成功率从约 88% 提升到接近 100%。

- **实现"碰撞检测 → 诊断报告 → LLM 修正 → 重新规划"的闭环自愈链路**。当轨迹在距离场上违反安全约束时，自动定位最差碰撞点的精确坐标与偏离量，生成结构化诊断信息回传给大模型，由大模型调整约束后 A* 绕行重新规划 —— 在受控测试中 3 轮内完成从碰撞到安全通过的自动修正。

**技术栈**：Python / Gradio / NumPy + SciPy / DeepSeek API / LibreDWG / OpenCV
